# TypeRacer-Hacks
A collection of hacks that can be used for TypeRacer.

This repository currently includes an automatic typer script (`autotype.js`) and a manual typer script (`manualtype.js`). More scripts may be added in the future.

## How to use

### autotype.js
**The automatic typer**
autotype.js is useful when you want to get consistent WPM on your profile without having to do much. Just a few key presses and you're off. Perfect for lazy people like me.

1. Copy the autotype.js script. Make sure to read and configure the contents of it. I recommend setting the WPM to something below **100 WPM** if you are not a fast typer.

2. Go into ANY race / practice mode / whatever.

3. Before the race begins, open up your Inspect Element / DevTools / F12 and move over to the Console tab.

4. Press **Ctrl+V.** If it asks, type **allow pasting** to make it allow you to paste the code in.

5. Double-check everything is set up how you want it to be. If you're ready, press Enter.

6. Close the Inspect Element / DevTools / F12. Make sure it's focused on the race.

7. Press **ANY LETTER** ONCE and immediately press Backspace. Enjoy!

**NOTE: YOU DO NOT NEED TO RE-ENTER THIS CODE EVERY TIME YOU PRESS RACE AGAIN AFTER A RACE! YOU ONLY NEED TO REDO IT IF YOU REFRESH YOUR PAGE.**

### manualtype.js
**The manual typer**
manualtype.js is a different version compared to autotype.js. Here, you have to manually press the keys yourself, but it always enters the correct one. This is better if you want to look more legitimate.

1. Copy the manualtype.js script.

2. Go into ANY race / practice mode / whatever.

3. Before the race begins, open up your Inspect Element / DevTools / F12 and move over to the Console tab.

4. Press **Ctrl+V.** If it asks, type **allow pasting** to make it allow you to paste the code in.

5. If you're ready, press Enter.

6. Close the Inspect Element / DevTools / F12. Make sure it's focused on the race.

7. Press **ANY LETTER** ONCE and immediately press Backspace. Every key you press after that will type the next correct character for you. Enjoy!

**NOTE: YOU DO NOT NEED TO RE-ENTER THIS CODE EVERY TIME YOU PRESS RACE AGAIN AFTER A RACE! YOU ONLY NEED TO REDO IT IF YOU REFRESH YOUR PAGE.**
